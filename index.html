eze/easy mode

github page

A userscript for e*hentai websites to add additional features

Installing

  1. For Firefox 57+: Install Greasemonkey 4+ or Tampermonkey
    For earlier versions: Use Greasemonkey 3.17

    For Chrome: Install Tampermonkey

  2. Install eze through the userscript manager

Features

  • Full title readability on search results

    Long, obscured titles can be properly read on mouse hover when using the thumbnail view

  • Search links on galleries

    Links for finding galleries by name / on nhentai
    This is sometimes useful for finding alternate versions

  • Removed gallery info recovery

    Removed gallery pages no longer redirect to the homepage
    Instead, they show information about the gallery recovered through the API

  • Gallery thumbnail loader

    Loads all rows of preview thumbnails onto a single page

  • Gallery downloader

    An easy way to download any gallery at no cost, which can be useful if no torrents exist
    Galleries can be downloaded as uncompressed .zip/.cbz and can include a JSON gallery info file
    Using full sized images will count more against the image limit, but there are options for this

  • Random gallery

    Click the button above the page navigation to go to a random gallery when searching

  • Login helper for sad panda

    Login instructions/links for easier bypassing of the sad panda

Troubleshooting

  • Make sure your script is up to date
  • Make sure your userscript manager is up to date
  • Make sure your browser is up to date
  • File a bug report if problems persist
